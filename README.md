# cgjb01 - 采购 2026 降本数据看板

这是一个基于 Streamlit 的采购降本台账数据看板，用于上传、解析和可视化《采购2026降本台账》Excel 文件。

## 功能

- 上传 `.xlsx` / `.xlsm` / `.xls` 台账文件
- 自动识别配件采购、跨境成品采购工作表
- 展示配件降本总额、跨境成品降本总额、供应商数、品号数
- 展示供应商降本贡献排行
- 展示月度降本趋势
- 检查 Q1 单价差异、数量差异
- 支持明细筛选、搜索和 CSV 导出

## 本地运行

```powershell
pip install -r requirements.txt
streamlit run src/app.py
```

## 数据解析规则

- 跳过 Excel 前两行表头，从第 3 行开始读取数据
- 使用绝对列索引解析，避免复杂合并表头导致匹配失败
- 供应商名称列会向下填充，以处理合并单元格
- Q1 对账检查列：
  - 1 月：单价差异 L 列，数量差异 O 列
  - 2 月：单价差异 S 列，数量差异 V 列
  - 3 月：单价差异 Z 列，数量差异 AC 列
